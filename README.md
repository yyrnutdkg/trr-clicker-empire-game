# trr-clicker-empire-game

## 概要

- 労働、投資、不動産を駆使してお金を稼いでいくゲーム

## ルール

- ユーザは 20 歳、ハンバーガー店員であり、1 秒で 1 日分時間が経過していく世界で働いている。
- 所持金は 50,000 円からのスタート。
- お金を貯めることで、他のアイテムを購入することができる。
- 現在購入できるアイテム一覧は下記の通り。
